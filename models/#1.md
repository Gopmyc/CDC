## 1. Contexte et Objectif de la Prestation

Le présent cahier des charges définit les éléments 3D à réaliser pour le jeu **Garry’s Mod**.
L’objectif est de produire des modèles 3D optimisés pour le moteur Source, avec ou sans animations, respectant une direction artistique low-poly et des textures haute résolution.

## 3. Style et Direction Artistique

Les modèles doivent respecter un style **low-poly**, avec des textures minimalistes mais en **haute résolution**.

### Caractéristiques spécifiques du style

* **Textures** : Béton avec **effets d’usure, variations naturelles du temps et impacts de balle sur les façades inférieures**.
* **Couleurs** : Teintes grises dominantes.
* **Animations** : N/A

---

## 4. Livrables attendus

Le prestataire devra fournir :

* **Fichiers sources** : BLEND, FBX ou OBJ.
* **Textures** : VTF (1024×1024 ou supérieur).
* **Animations** : Si applicable, formats compatibles Source Engine.
* **Fichiers exportés** optimisés pour Garry’s Mod.

### Formats à livrer pour un modèle GMod

| Type de fichier    | Format requis                  | Description                                          |
| ------------------ | ------------------------------ | ---------------------------------------------------- |
| **Modèle source**  | `.FBX`, `.OBJ`, `.DMX`, `.SMD` | Formats d’export pour compilation vers Source Engine |
| **Modèle compilé** | `.MDL`, `.VTX`, `.PHY`, `.ANI` | Formats finaux pour le moteur Source                 |
| **Textures**       | `.VTF`                         | Texture Source Engine                                |
| **Matériaux**      | `.VMT`                         | Fichiers de shaders et propriétés textures           |
| **Animations**     | `.SMD`, `.DMX`, `.ANI`         | Pour les animations de modèles (si applicable)       |

* Respect des **dimensions et formats**.
* Cohérence avec la **direction artistique**.
